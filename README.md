# Kazakh Folklore

A React-based website dedicated to Kazakh folklore and mythological creatures.

## About
This project showcases characters from Kazakh folk tales, including legendary 
figures such as Zheztyrnak, Mystan-Kempir, and Zhalmauyz-Kempir.
Features a parallax scrolling welcome page and an interactive carousel of characters.

## Tech Stack
- React
- React Bootstrap
- React Multi Carousel
- Animate.css
- JavaScript (ES6+)

## Getting Started
```bash
npm install
npm start
```
